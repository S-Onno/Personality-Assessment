# Personality-Assessment

#概要：
学園祭のクラス出し物で使用するシステムの開発「性格診断アプリケーション」

MBTIみたいに12タイプに分けて、それを4つのグループにわけます



## 性格のグループ

### インフラ型　(分析家)

- オンプレミスタイプ　(黙々作業)
- クラウドタイプ　    (フットワーク・気さく)
- ハイブリッドタイプ　(仲介人)

### エンジニア型　(探検家)

- フロントエンドタイプ (リーダーシップ)
- バックエンドタイプ　(縁の下の力持ち)
- フルスタックタイプ　(神様)

### セキュリティ型　(番人)

- ファイアウォールタイプ　(擁護者)
- マルウェアタイプ　(裏切者)
- ゼロトラストタイプ　(一匹狼)

### ネットワーク型　(外交官)

- ルータータイプ　(仲介者)
- 光ファイバータイプ　(エンターテイナー)
- スイッチタイプ　(コミュニケーション能力が高い)
